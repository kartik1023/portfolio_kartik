## 🎨 Customization Guide

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-bg: #0A192F;
    --primary-color: #007BFF;
    --secondary-color: #00C2FF;
    /* ... more colors */
}
```

### Update Content
Edit the HTML in `index.html`:
- Personal information in the hero section
- Experience details in the timeline
- Project descriptions in the projects section
- Skills in the skills section

### Add Projects
Copy a project card in the projects section and update:
- Project title
- Description
- Technology tags
- Links
